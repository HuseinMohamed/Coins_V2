# مولّد كارت العملات الاحترافي

برنامج Streamlit: بترفع صور عملة قديمة (وجه وظهر)، والبرنامج بيكتب نبذة تاريخية عنها (باستخدام Groq Vision)
وبيولد كارت احترافي جاهز للنشر على السوشيال ميديا — إما "كارت بسيط" بألوان قابلة للتخصيص، أو "بوستر فاخر"
بخلفية صورة جاهزة.

## محتويات الفولدر
- `app.py` — البرنامج نفسه
- `requirements.txt` — المكتبات المطلوبة
- `Amiri-Regular.ttf` — خط عربي (لازم يفضل في نفس الفولدر)
- `logo.png` — لوجو المحل
- `whatsapp_icon.png` — أيقونة الواتساب
- `address_icon.png` — أيقونة العنوان
- `poster_bg.png` — خلفية تصميم البوستر الفاخر
- `.streamlit/secrets.toml.example` — قالب لحفظ مفتاح Groq API

كل الملفات دي (الخط، اللوجو، الأيقونات، خلفية البوستر) لازم تفضل بنفس الاسم في نفس فولدر `app.py`.

## التشغيل

```bash
pip install -r requirements.txt
streamlit run app.py
```

هيفتحلك البرنامج في المتصفح تلقائيًا (عادة على http://localhost:8501).

## نوعين تصميم
- **كارت بسيط**: خلفية لونية (٥ ثيمات جاهزة)، فيه مكان لنبذة تاريخية (يدوي أو تلقائي بالذكاء الاصطناعي).
- **بوستر فاخر**: خلفية صورة جاهزة (`poster_bg.png`) بشكل إعلاني احترافي، مع شريط علوي وسطر فرعي وسعر بارز.
  تقدر تغيّر نص الشريط العلوي والسطر الفرعي من الفورم، أو تسيبهم فاضيين وهياخدوا نص افتراضي.

## مفتاح Groq API (لكتابة النبذة تلقائيًا — وضع "كارت بسيط" بس)

### للتشغيل على جهازك (يتقرا أوتوماتيك)
1. اعمل نسخة من `.streamlit/secrets.toml.example` وسمّيها `secrets.toml` (من غير `.example`)
2. حط مفتاحك الحقيقي بدل النص:
   ```toml
   GROQ_API_KEY = "gsk_مفتاحك_الحقيقي_هنا"
   ```
3. شغّل البرنامج، هتلاقي في الشريط الجانبي "مفتاح Groq API مفعّل تلقائيًا ✓"

### لما ترفعه أونلاين (Streamlit Community Cloud)
**متحطش المفتاح في ملف بترفعه على GitHub.** بدل كده:
1. بعد نشر التطبيق، روح Settings > Secrets
2. الصق نفس السطر: `GROQ_API_KEY = "gsk_مفتاحك_الحقيقي_هنا"`

ملف `.gitignore` المرفق بيمنع `secrets.toml` الحقيقي من الرفع بالغلط على GitHub.

اعمل حساب مجاني على console.groq.com لو مالكش مفتاح. ملحوظة: الموديلات اللي بتقدمها Groq بتتغير بمرور
الوقت، فلو ظهر خطأ "model not found"، حدّث اسم الموديل في دالة `get_coin_story` داخل `app.py`.

## المميزات
- رفع صورتين (وجه وظهر)؛ الظهر اختياري
- نوعين قطعة: معدنية (برواز دائري) أو ورقة نقدية (برواز مستطيل) — بينطبق على التصميمين
- نوعين تصميم: كارت بسيط (٥ ثيمات ألوان) أو بوستر فاخر (خلفية جاهزة)
- لوجو المحل يظهر تلقائيًا
- منطقتين نص حر بجانب أيقونات واتساب وعنوان اختيارية
- نبذة تاريخية يدوي أو تلقائي بالذكاء الاصطناعي (وضع الكارت البسيط)
- ارتفاع الكارت البسيط بيتظبط تلقائيًا حسب طول المحتوى

## ملاحظات
- النبذة التاريخية اللي بيكتبها الذكاء الاصطناعي تقدير عام، مش تحديد علمي دقيق — راجع خبير نوميزماتيك للتأكد.
- عايز تضيف ثيم لون جديد للكارت البسيط؟ افتح `app.py` ودور على `THEMES = {...}`.
- عايز تغيّر تفاصيل تصميم البوستر (أحجام، مسافات، ألوان)؟ كل حاجة في دالة `build_poster`.
